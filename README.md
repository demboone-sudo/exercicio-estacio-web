# Exercício de Desenvolvimento Web - Estácio 🚀

Este repositório contém as atividades práticas das aulas 1, 2 e 3 da disciplina de Desenvolvimento Web. O foco do projeto foi a estruturação de documentos HTML, uso de tabelas, semântica e estilização básica com CSS.

## 💻 Tecnologias Utilizadas
* **HTML5**: Estruturação de conteúdo e tabelas semânticas.
* **CSS3**: Estilização visual (cores, fontes, espaçamentos).
* **Git & GitHub**: Versionamento de código e hospedagem.
* **Notepad++**: Editor de texto utilizado no desenvolvimento.

## 📋 O que foi desenvolvido
- **Página Inicial (Index)**: Resumo do progresso e links de navegação.
- **Aula 1**: Introdução às tags básicas.
- **Aula 2**: Tabelas simples para organização de dados.
- **Aula 3**: HTML Semântico (`header`, `main`, `footer`) e tabelas estruturadas (`thead`, `tbody`, `tfoot`).
